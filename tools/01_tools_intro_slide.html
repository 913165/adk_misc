core concept

Tools are just Python functions

def track_order(order_id: str) -> dict:
    """Looks up the status of a customer order.

    Args:
        order_id: The order ID like "ORD-1001".
    """
    orders = {
        "ORD-1001": "Shipped, arrives April 14",
        "ORD-1002": "Processing",
    }
    return {"status": orders.get(order_id, "Not found")}
  • Type hints tell the LLM what arguments to pass
  • Docstring tells the LLM when and why to call it
  • Return value goes back to the LLM as context
  • No decorators — just pass the function directly
# Wire it up to an agent
agent = Agent(
    name="order_agent",
    model="gemini-2.5-flash",
    instruction="Use track_order to look up orders.",
    tools=[track_order],  # ← that's it
)
